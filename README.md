# 🔐 Password Manager (Python POO)

Sistema simples de gerenciamento de senhas com criptografia.

## 🚀 Funcionalidades
- Adicionar senhas por serviço
- Buscar senhas com descriptografia
- Listar serviços
- Remover senhas
- Armazenamento em JSON

## 🔒 Segurança
As senhas são criptografadas usando Fernet (cryptography).

## ▶️ Como rodar

```bash
pip install -r requirements.txt
python main.py
